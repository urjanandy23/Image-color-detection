Image Color Detection Project

This project implements an image color detection system using Python.
It allows users to upload an image and automatically detects the
dominant colors present in that image.

Features

-   Upload and process images.
-   Detect dominant colors using clustering.
-   Visualize results.
-   Google Colab Notebook for easy execution.

Files Included

-   image_color_detection_colab.ipynb — Main Google Colab notebook.

Requirements

-   Python 3.x
-   Libraries:
    -   numpy
    -   pandas
    -   matplotlib
    -   sklearn
    -   cv2 (OpenCV)

How to Use

1.  Open the .ipynb file in Google Colab.
2.  Upload an image when prompted.
3.  Run all cells to see the detected colors.
